# AirHockey
Mobile sports game made with Unity 6000.2.10f which allows you tu play against computer or another player in hotseat mode.

Root folder contains game installation file AirHockey.apk and Unity 6000.2.10f project folder "Project".
Abstraction and inheritance are used as basis for game architecture to keep it's classes as modular and isolated as possible to uphold
good coding rules and principles.
